## 1.1 代码提交步骤

初次时，本地执行如下代码

```shell
git init
git add README.md
git commit -m "first commit"
git branch -M master
git remote add origin git@github.com:lm1611944656/Test.git
git push -u origin master
```

或者

```shell
git remote add origin git@github.com:lm1611944656/Test.git
git branch -M master
git push -u origin master
```
